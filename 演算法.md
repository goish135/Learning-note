<h4 id="2">0/1背包問題</h4>

[參考網址](https://openhome.cc/Gossip/AlgorithmGossip/KnapsackProblem.htm)

設有ABCDE五種水果，要放入負重為8kg的背包，求該放進哪些水果可得最佳解?

| index | 種類 | 公斤 | 價格 |
| - | - | - | - |
| 0 | A | 4kg | 4500 |
| 1 | B | 5kg | 5700 |
| 2 | C | 2kg | 2250 |
| 3 | D | 1kg | 1100 |
| 4 | E | 6kg | 6700 |

sol:<br>
一個元素一個元素依序放入A->B->C->D->E,並假設有1~8kg的背包8個<br>
求各階段(放入第?th個水果時，各個負重量)的最佳解。<br>
value[8+1]: 紀錄各負重的價值，items[8+1]:紀錄各負重最後放進水果的index <br>
```C++
#include <stdio.h> 
#include <stdlib.h> 
#define LIMIT 8   // 重量限制 

typedef struct { 
    char name[20]; 
    int weight; 
    int price; 
} Fruit; 

void knapsack(Fruit*, int*, int*, int, int);
int min(Fruit*, int);

int main(void) { 
    Fruit fruits[] = {
                      {"李子", 4, 4500},
                      {"蘋果", 5, 5700},
                      {"橘子", 2, 2250},
                      {"草莓", 1, 1100},
                      {"甜瓜", 6, 6700}
                     };
    int items[LIMIT + 1] = {0}; 
    int values[LIMIT + 1] = {0};  
    
    int length = sizeof(fruits) / sizeof(fruits[0]);
    knapsack(fruits, values, items, length, LIMIT);

    printf("物品\t價格\n"); 
    int i;
    // min(fruits, length) :最輕的水果 
    for(i = LIMIT; i >= min(fruits,length) ; i -= fruits[items[i]].weight) {
        printf("%s\t%d\n", fruits[items[i]].name, fruits[items[i]].price); 
    } 
    printf("合計\t%d\n", values[LIMIT]); 

    return 0; 
}  

void knapsack(Fruit* fruits, int* values, int* items, 
              int length, int limit) {
    int i, w;
    for(i = 0; i < length; i++) { 
        for(w = fruits[i].weight; w <= limit; w++) {
            int p = w - fruits[i].weight;
            int newValue = values[p] + fruits[i].price; 
            if(newValue > values[w]) {   // 找到階段最佳解 
                values[w] = newValue; 
                items[w] = i; 
            }
        } 
    }
}

int min(Fruit* fruits, int length) {
    int i, m;
    for(i = 0, m = fruits[0].weight; i < length; i++) {
        if(fruits[i].weight < m) {
            m = fruits[i].weight;
        }
    }
    return m;
}
```

<h4 id="6">Dijkstra</h4>

[原理](http://nthucad.cs.nthu.edu.tw/~yyliu/personal/nou/04ds/dijkstra.html)<br>
***類題*** >> NCPC2015 Problem B <br>

解法:因為是一個無向圖，(1)將"會付錢的"的station 所連接的道路的花費扣除(扣完可能是負數)，<br>
(2)再將"起始站所連接的"道路的權重扣除一開始給的錢。<br>
最後 結合 Dijkstra 的原理，先找出離原點__最近的節點__，如果為正(需要花費)，但一開始已先扣除前面兩種情況,直接terminate,印出答案。<br>
如果為負數或零，代表可繼續走下去(再找出花費最少的下一個)，新增__該節點__，並更新**經**該節點，可使原點離其他點更近的**距離**。<br><br>
<ps>無窮大inf設0x3f3f3f[見原因](http://blog.csdn.net/L_apple8/article/details/52525752)<br>


```C++
#include<stdio.h>
#include<cstring>
#include<algorithm>
using namespace std;
#define inf 0x3f3f3f
#define MAX 32
int i_c,s_s;
int getm[MAX];
int dis[MAX][MAX] = {0};

void dijkstra(int n)
{
    int len[n],used[n];
    
    for(int i=1;i<=n;i++)
    {
        len[i]=dis[s_s][i];
        used[i]=0;
    }
    
    while(1)
    {
        int mi=inf;
        int newn=-1;
        
        for(int i=1;i<=n;i++)
        {
            //printf("len[%d]:%d\n",i,len[i]);
            if(mi>len[i]&&used[i]==0)
            {
                mi=len[i];
                newn=i;
            }
        }
        
        //printf("newn:%d\n",newn);
        
        if(len[newn]>0||newn==-1)
            break;
        
        used[newn]=1;
        
        
        for(int i=1;i<=n;i++)
        {
            if(len[i]>dis[newn][i]+len[newn])
                len[i]=dis[newn][i]+len[newn];
        }

    }
    
    int ans=0;
    for(int i=1;i<=n;i++)
    {
        if(!used[i])
            ans++;
    }
    
    printf("%d\n",ans);
}
int main()
{
    //freopen("PB.in","r",stdin);
    int tc;
    scanf("%d",&tc);
    while(tc--)
    {
        memset(dis,inf,sizeof(dis));
        memset(getm,0,sizeof(getm));
        int n_s;
        scanf("%d%d%d",&n_s,&s_s,&i_c);
        int p; //has service 
        scanf("%d",&p);
        int s1,s2,c;
        for(int i=0;i<p;i++)
        {
            scanf("%d%d%d",&s1,&s2,&c);
            dis[s1][s2]=c;
            dis[s2][s1]=c;
        }
        int z; // pay money station 數
        scanf("%d",&z);
        int s,get;
        for(int i=0;i<z;i++)
        {
            scanf("%d %d",&s,&get);
            getm[s]=get;
            for(int j=1;j<=n_s;j++)
            {
                dis[s][j]-=getm[s];
            }
        }
        for(int j=1;j<=n_s;j++)
        {
            dis[s_s][j]-=i_c;
        }
        
        /*for(int i=1;i<=n_s;i++)
        { 
            for(int j=1;j<=n_s;j++)
            { 
                printf("%d ",dis[i][j]);    
            }
            printf("\n");
        }*/
        
        dijkstra(n_s);
    }
}
```
